# Smart Library Management System

This is the backend foundation for a team project designed as a modular and structured Library Management System built in Java.

## 📌 Project Overview

This system handles:
- Book add
- View all books
- Issue book
- Return book
- Search books by title
- Data persistence using text file storage

Built in a way that supports further expansion with Servlets, JDBC, and Spring MVC.
